# git-documentation

```sh
git --version
```

Perintah ini berguna untuk mengetahui versi dari git kita

## Git Configuratoin
```sh
git config --global user.name "namau"
git config --global user.email "urimel"
```

## Make git Folder
```sh
mkdir (nama folder)
```

## Git Initialize
```sh
git init
```

## Check Status
```sh
git status
```
